Software Systems

These are the sample programs that I made using C# and Mysql.

Dream Shakes Inventory system

This is an Ordering System with an AI Assistant that uses Speech recognition. This application was built with the help of C# and MYSQL. This project is made especially for persons with visual problems or an arm amputee that want to order a product in an automated machine. This system will make their lives esier because they only need to speak in order to purchase a product.





Tagay's Inventory System

This application is an ordering system that was made especially for a small Shake store located in Tanay, Rizal. This application was made to help the business to figure what are the most demanding products on their shop so shortage of best sellers will prevent. It will also help them to improve those products that are not that much favorite of the customers.
